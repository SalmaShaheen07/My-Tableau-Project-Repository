# My-Tableau-Project-Repository
## Project Summary
This repository contains my Tableau visual analytics project completed during the Data Technician Bootcamp (May–June 2026).
The project focuses on exploring global health, retail, and sales‑related datasets using Tableau to build interactive dashboards and communicate insights through data storytelling.

Across multiple tasks, I created visualisations, dashboards, and analytical views that demonstrate core Tableau skills used in real‑world data roles.

## Project Overview
Throughout the Tableau module, I worked with datasets such as:

Gapminder Health Data

Spotify Track Features (optional extension task)

Additional retail and sales‑related datasets

The goal was to explore trends, compare metrics, and build interactive dashboards that support decision‑making.

## Key Tableau Skills Demonstrated
### Interactive Dashboards
I designed dashboards combining multiple worksheets, enabling users to:

Filter by country, continent, or category

Interact with slicers and dropdowns

Explore linked visualisations across multiple views

Navigate insights through a clean, user‑friendly layout

### Charts & Visualisations
I created a variety of chart types to answer analytical questions:

Bar charts – e.g., Life Expectancy by Continent

Line charts – e.g., Life Expectancy trends over time

Pie charts – e.g., Population distribution by gender

Scatter plots – e.g., Life Expectancy vs BMI

Maps (where applicable) – for geographic comparisons

Custom visual of choice – exploring additional metrics

These visuals helped highlight disparities, trends, and relationships across countries and health indicators.

### Calculated Fields & Data Transformations
I applied Tableau calculations to enhance analysis, including:

Average Life Expectancy

Top N filters (e.g., Top 5 countries by life expectancy)

Percentage of total (for pie charts)

Custom metrics for deeper comparisons

### Filters, Slicers & Interactivity
To improve user experience, I used:

Single‑value dropdown filters

Global filters applied across multiple worksheets

Continent and Country slicers

Interactive legends for highlighting and isolating data

These features allowed users to explore the dashboard dynamically.

### Project Deliverables
✔️ Life Expectancy by Continent
A bar chart showing average life expectancy for each continent in the most recent year.

✔️ Life Expectancy Trend (Top 5 Countries)
A line chart comparing life expectancy over time for the top 5 countries.

✔️ Population by Gender
A pie chart showing gender distribution for a selected country in the latest year.

✔️ Life Expectancy vs BMI
A scatter plot exploring the relationship between BMI and life expectancy.

✔️ Custom Visualisation
An additional chart exploring another health or demographic metric.

✔️ Global Health Dashboard
A combined dashboard titled “Global Health Insights”, published to Tableau Public.

This project strengthened my ability to:

Build interactive dashboards for real‑world decision‑making

Communicate insights visually through storytelling

Compare trends across countries, continents, and demographic groups
